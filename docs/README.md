# docs/

Reporte gráfico en LaTeX para la evaluación.

## Contenido esperado

- `main.tex` — Documento principal.
- `sections/` — Capítulos:
  - `introduccion.tex`
  - `arquitectura.tex`
  - `implementacion.tex`
  - `pruebas.tex`
  - `conclusiones.tex`
- `figures/` — Diagramas de red, capturas de pantalla del panel.
- `Makefile` — Compilación con `latexmk`.

## Compilar

```bash
cd docs
make
```

Genera `main.pdf`.
