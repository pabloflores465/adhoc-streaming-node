\chapter{Introducción}

\section{Objetivo}
Desarrollar una red AD-HOC inalámbrica capaz de transmitir streaming de audio en tiempo real entre múltiples nodos que operan desde USB con GNU/Linux.

\section{Alcance}
\begin{itemize}
    \item Boot desde USB con Ubuntu Server.
    \item Auto-deteción de redes AD-HOC existentes.
    \item Creación dinámica de redes cuando no hay cobertura.
    \item Elección de Master por recursos del sistema.
    \item Streaming multicast de audio vía UDP.
    \item Panel de indicadores en tiempo real.
\end{itemize}
