\chapter{Conclusiones}

\begin{itemize}
    \item La red AD-HOC logró auto-configuración sin infraestructura central.
    \item El streaming multicast sobre UDP redujo tráfico redundante respecto a unicast.
    \item La elección de Master por recursos distribuye la carga de manera dinámica.
    \item El panel web proporcionó visibilidad completa del estado de red en tiempo real.
    \item Se cumplieron todos los requerimientos funcionales establecidos en la rúbrica.
\end{itemize}
